# Push-уведомления для администратора

## Что нужно
1. Service Worker уже есть (sw.js)
2. Нужно: `Notification.requestPermission()` + `new Notification()` при WS-событиях
3. Push API требует VAPID ключи + backend — пока не нужно

## Решение (без Push API)
Браузерные уведомления через `Notification` API:
- При `gate:request` → `new Notification('Запрос на открытие ворот', { body: 'Домик #N' })`
- При `message:new` → `new Notification('Новое сообщение', { body: 'Домик #N' })`
- При `task:new` → `new Notification('Новая заявка', { body: 'Тип - Домик #N' })`
